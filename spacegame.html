Untitled Space Game

A topdown 2d retro game prototype that incorperates cozy gameplay and bullet hell combat.

Development

December 2024

I just started development on this prototype (not really a game yet) to see if it works.
The idea is to spark creativity with the cozy elements that will be base design, but also challenge the player
in both player vs player and player vs bot combat and try to inspire "world" exploration of
interstellar procedurally generated retro space filled with loot and fights.

The game is currently being developed with audio too which features
some tracks made by my friend Lashlie.

Preview

The prototype draws inspiration from the space game Faster Than Light (FTL), Disney movie Treasure Planet (Disney) and other medias set in space.
